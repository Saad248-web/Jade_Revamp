6. Env · database · integrations (operators)

See project .env.example for full keys. Highlights:

  • PostgreSQL: POSTGRES_* → src/lib/db.ts
  • Admin: ADMIN_PASSWORD (x-admin-password header)
  • Mail: RESEND_API_KEY, RESEND_FROM, LEADS_NOTIFY_EMAIL, BOOKING_NOTIFY_EMAIL, CAREERS_NOTIFY_EMAIL
  • Razorpay: NEXT_PUBLIC_PAYMENT_GATEWAY_KEY, RAZORPAY_KEY_*, RAZORPAY_WEBHOOK_SECRET
  • Sentry (optional): NEXT_PUBLIC_SENTRY_DSN · @sentry/nextjs + src/instrumentation*.ts
  • IndexNow: INDEXNOW_* + Bearer INDEXNOW_API_SECRET
  • Analytics: NEXT_PUBLIC_GA_ID → GoogleAnalytics in providers.tsx
