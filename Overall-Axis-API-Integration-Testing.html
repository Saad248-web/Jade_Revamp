API 2 runs after successful API 9 (HTTP 200). Validation failures (422/401) do not push inventory.

5. Critical Validation Gate (API 9)

  1. Invalid accessKey → 401
  2. Bad dates / noOfRooms != 1 → 422 (distinct message)
  3. Unknown hotelId+roomId or bad ratePlanId → 422 (distinct message)
  4. Save booking + API 2 inventory ack → 200
